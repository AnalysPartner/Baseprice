Purchasing tool for Swisslog

Hi! This tool contains two main functions and then one sub function.
1: Cleans PIR data directly from SAP, you input the file you recieve from SAP after running the ZMMINFORECEXT transaction and the output is a Excel workbook with only fixed suppliers.
2: Compares prices, suppliers and leadtime between the preconfigured baseprice list and the PIR-list.

Function one: Clean PIR


Press “Clean raw PIR”
Select your file that you got from SAP
Select where you want to save the file
wait for prompt “Script Complete”

Function two: Analyse


Press “Run analysis”
Select your now cleaned version of PIR
!Important, make sure the excel workbook you select only contains fixed suppliers!
Select where you want to save the file
wait for prompt “Script Complete”

Sub function: Manage database


Press “Manage database”

Password is Swisslog, this is just a safety precaution to avoid misstakes.
Now you have multiple options.
Print
These are to bottom row buttons, select which record you want to print then select save location and the output is a Excel workbook in the correct format.
Upload new
Press the “Activate buttons” button and then press ok on the prompt. (this is also just for safety to avoid misstakes)
Select what record you want to upload.
Note1: Good practice is to first make a copy by printing out old records.
Note2: When you want to update a newer version of the records print the old one and just update the values, it is very important that the format stays the same.
